# ARPG Automation Scripting Engine

Инструмент для автоматизации игровых механик (макросов) в Action RPG играх. Позволяет гибко настраивать последовательности нажатий через внешние конфигурации.

## Ключевые особенности
* **Архитектура на ООП:** Использование базового класса `BaseScript` для легкого расширения функционала новыми типами скриптов.
* **Многопоточность:** Выполнение макросов вынесено в отдельные потоки (`threading`), что исключает зависание основной программы.
* **Гибкая конфигурация:** Все параметры (клавиши, задержки, горячие клавиши) загружаются из `params.json` без необходимости менять код.
* **Безопасность (Debounce):** Реализована защита от случайных двойных нажатий и система блокировки горячих клавиш.

## Стек технологий
* **Language:** Python 3.x
* **Libraries:** `keyboard`, `threading`, `json`

## Структура проекта
* `base.py` — Базовый класс с логикой защиты от дребезга и блокировкой.
* `logic.py` — Конкретные реализации макросов (Scripter и Togler).
* `main.py` — Точка входа, отвечающая за загрузку конфигов и инициализацию.

## Как использовать
1. Настройте нужные клавиши в `params.json`.
2. Запустите `main.py`.
3. Используйте заданные горячие клавиши для активации скриптов. Остановка всех процессов — `F12`.
